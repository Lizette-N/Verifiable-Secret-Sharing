# Verifiable Secret Sharing
This repository contains Python prototypes developed as part of our bachelor project on Verifiable Secret Sharing.
The code is intended as a local prototype for testing and understanding the main protocol flow. 

## Repo Structure
- `SynchronousVSS.py`  

  Prototype implementation of the synchronous VSS protocol.

- `AsynchronousVSS.py`  

  Prototype implementation of the asynchronous VSS protocol.

- `PC.py`  

  Simplified Pedersen-style polynomial commitment scheme.

- `Signatures.py`  

  Signature helper functions.

  ## Requirements
  Recommended python 3.10 or newer.
  Python cryptography
